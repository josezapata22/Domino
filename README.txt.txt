Bienvenido a mi proyecto sobre un juego de domino.
Para empezar la consola comienza preguntando cual es el nombre del primer jugador, lo escribes y le das a enter
cuando lo hagas saldra una lista de las posiciones disponibles en la mesa, selecciona usando los numeros que le corresponden
, se repite asi con los demas jugadores, luego comienza el juego, se uega seleccionando el indice de la ficha correspondiente para jugar
